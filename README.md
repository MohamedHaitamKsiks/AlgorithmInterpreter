# Algorithm Interpreter
Algorithm interpreter written using python.

## The language supports:
FOR and WHILE loops
Conditions
Static Types Variables
Console I/O

## Code example

    VAR
        k,nombre,f : ENTIER;
    
    DEBUT
        f<-8;
        nombre<-10;
        
        POUR k DE 1 A nombre PAS 1 FAIRE
            f<-f+k;
        FINPOUR
    
        ecrire(f);
    FIN
